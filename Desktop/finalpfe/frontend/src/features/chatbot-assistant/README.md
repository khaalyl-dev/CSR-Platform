# ChatbotAssistant (Optional)

Assistance interactive pour aider les utilisateurs.

## Scope

- Réponses aux questions fréquentes sur le plan CSR, validation et reporting
- Notifications et guidance sur les workflows

## Structure

- `models/` – ChatbotLog
- `api/` – chatbot-api

## À développer

- [ ] **Chatbot API** – Envoyer question, récupérer réponse, log chatbot_logs
- [ ] **Widget chat** – Composant fenêtre chat flottante
- [ ] **FAQs** – Base de questions/réponses (plan CSR, validation, reporting)
- [ ] **Contexte** – Passage site_id, user_id pour réponses contextualisées
- [ ] **Historique** – Affichage des échanges récents (optionnel)
